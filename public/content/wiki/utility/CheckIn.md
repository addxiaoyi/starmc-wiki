# 日常签到福利

每天上线签到即可领取基础奖励，坚持连续签到还能获得更丰厚的资源和点券。

## 📅 签到机制
- **参与方式**：输入 `/signin` 或在主菜单点击签到图标即可。
- **连续奖励**：
  - **3天**：额外奖励 1,500 金币。
  - **7天**：额外奖励 8,000 金币 + 80 点券 + **1张补签卡**。
  - **15天**：额外奖励 20,000 金币 + 200 点券 + **2张补签卡**。
  - **25天**：赢取终极“舵星锦鲤”大奖。

## 🔄 补签说明
如果不小心断签了，可以使用补签卡恢复进度。
- **补签卡获取**：通过连续签到 7 天或 15 天自动获得。
- **查看余额**：输入 `/signin info` 查看当前拥有的补签卡数量。
- **操作限制**：补签有 5 秒的冷却间隔，请耐心操作。

## ⌨️ 常用指令
| 指令 | 功能 |
| :--- | :--- |
| `/signin` | 打开签到主界面 |
| `/signin info` | 查看个人签到统计与补签卡 |

::: tip 小提示
签到前请确保背包有足够空位，否则部分实物奖励可能无法正常发放。
:::
