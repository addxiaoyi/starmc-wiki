# 每日挑战任务

在 StarMC 的日常生存中，通过完成每日挑战任务来赚取金币和各种稀有奖励。

## 🎯 任务体系
任务系统每天都会为你随机分配不同难度的挑战：

- **简单 (Easy)**：每天 4 个，适合上线顺手完成，奖励基础金币。
- **中等 (Medium)**：每天 3 个，需要花一点时间，奖励更为丰厚。
- **困难 (Hard)**：每天 2 个，具有一定的挑战性，但回报也非常可观。

## ⚙️ 核心机制
- **自动重置**：任务在每日 **00:00** 准时刷新，请记得在刷新前领取奖励。
- **实时追踪**：当你完成任务步骤时，聊天框会弹出进度提示。
- **可视化菜单**：输入 `/quests` 即可打开图形界面，随时查看任务要求和剩余时间。
- **登录提醒**：每日上线 5 秒后，系统会自动提醒你今日的任务进度。

## ⌨️ 常用指令
| 指令 | 功能 |
| :--- | :--- |
| `/quests` | 打开每日挑战任务菜单 |

::: tip 小提示
完成任务获得的积分可以在任务商店兑换稀有物资，别忘了经常去看看。
:::
