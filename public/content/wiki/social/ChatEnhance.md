# 聊天增强功能

StarMC 拥有强大的交互式聊天系统，你可以直接在聊天框中分享物品属性、背包内容甚至实时坐标。

## 📦 快捷展示
在聊天框输入以下占位符，即可向全服玩家展示你的实时状态：

- `[item]` 或 `[hand]`：展示你手中持有的物品及其详细属性。
- `[inv]` 或 `[i]`：展示你的整个背包库存。
- `[ender]` 或 `[ec]`：展示你的末影箱内容。
- `[pos]`：分享你当前的精确坐标。

## 🔔 精准提醒
使用 `@玩家名` 提及他人时，对方会收到清脆的提示音和屏幕大标题提醒，确保信息不被错过。

## ✨ 智能交互
- **悬停名片**：将鼠标悬停在聊天框的玩家名上，可以查看对方所在的维度、实时血量和网络延迟。
- **点击操作**：点击玩家名可快速发起私聊，点击坐标则可以直接开启导航。
- **彩色聊天**：支持 RGB 颜色代码，让你的消息更具个性。

::: tip 小提示
想要向大家炫耀你的极品武器？只需在聊天中输入 `[item]` 即可，大家将鼠标悬停在信息上就能看到完整属性。
:::
