 # 聊天增强功能

如果你喜欢边玩边交流，这页会很实用。服务器的聊天不只是发文字，还能直接展示物品、坐标和更多状态信息。

## 先看结论

这套聊天功能最适合三种场景：

- 和朋友分享装备
- 发坐标找人或约集合
- 交易时快速确认物品

## 直接展示什么

### 展示手上物品

输入 `[item]` 或 `[hand]`，就能把你手里拿着的物品展示出来。

适合：

- 炫耀装备
- 交易前确认物品
- 发给朋友看属性

### 展示背包

输入 `[inv]` 或 `[i]`，可以展示背包内容。

适合：

- 交易前核对物品
- 临时给别人看背包
- 让队友帮你确认物资

### 展示末影箱

输入 `[ender]` 或 `[ec]`，可以展示末影箱内容。

适合：

- 整理存储
- 交易前确认稀有物品
- 做物资清点

### 分享坐标

输入 `[pos]`，可以直接分享当前位置。

适合：

- 找路
- 叫朋友过来
- 报坐标求助

## 提到其他玩家

输入 `@玩家名`，对方会收到提醒。

适合：

- 叫人看消息
- 找队友
- 提醒交易对象

## 聊天里还能做什么

### 悬停查看信息

把鼠标放到玩家名上，可以看到一些额外信息，比如：

- 所在维度
- 当前血量
- 网络延迟

这对聊天沟通和找人很方便。

### 点击快速操作

部分聊天内容可以直接点击，例如：

- 点击玩家名发起私聊
- 点击坐标快速导航

这能减少重复输入。

### 使用颜色和样式

如果服务器允许，你也可以用颜色代码让聊天更醒目。不过建议以清晰为主，不要把聊天搞得太乱。

## 日常最常用的几个场景

- 想发装备给别人看：`[item]`
- 想叫朋友过来：`[pos]` + `@玩家名`
- 想确认交易内容：`[inv]` 或 `[ec]`
- 想私聊：点玩家名或用 `/msg`

## 使用建议

- 发坐标时顺手说明一下你在干什么
- 交易前先确认双方都看到了同一个物品
- 不要只发一串占位符，最好带一点说明文字
- 和陌生人沟通时，尽量简洁、明确

## 一个更像玩家的用法

可以这样发：

```text
我在这里，坐标 [pos]，来领地看看。
```

或者：

```text
这是我刚做好的武器：[item]
```

这样别人一看就知道你想表达什么。
