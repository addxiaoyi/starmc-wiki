# 动作与肢体交互

StarMC 提供了多样化的动作表现，让你的角色更加鲜活。可以随时坐下、躺下，甚至与其他玩家进行趣味互动。

## 🛋️ 随处可坐
通过点击方块底部快速坐下：
- **楼梯与台阶**：所有木质、石质的楼梯和台阶。
- **柔软表面**：地毯、苔藓地毯或雪层。
- **操作要求**：**主手必须为空**，且距离方块较近。

## 👥 玩家互动
- **叠罗汉**：右键点击其他玩家，可以坐在对方肩膀上。
- **NPC 互动**：同样支持坐在服务器内的 NPC 身上。
- **拍一拍**：右键点击其他玩家触发“拍一拍”互动，带有趣味粒子和音效。

## 🤸 姿态控制
- **躺下休息**：输入 `/lay` 让角色原地躺下（还可以重置幻翼生成计时）。
- **爬行/旋转**：使用 `/crawl` 进入爬行状态，`/spin` 可以在原地旋转。
- **恢复姿态**：坐下或躺下后，按 **Shift (潜行)** 即可恢复。

## ⌨️ 常用指令
| 指令 | 功能 |
| :--- | :--- |
| `/sit` | 强制原地坐下 |
| `/lay` | 原地躺下 |
| `/crawl` | 进入爬行状态 |
| `/patpat` | 打开“拍一拍”互动设置菜单 |

::: tip 小提示
如果你不想被他人“拍一拍”或骑乘，可以在个人设置菜单中开启免打扰模式。
:::
