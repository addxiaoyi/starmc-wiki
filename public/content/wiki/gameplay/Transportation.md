 # 城市交通网络 
 
 服务器提供全自动化的城市交通网络，通过地铁系统连接各个主要区域。 
 
 ## 章节一：基础介绍 
 您可以乘坐全自动地铁穿梭于服务器的各个站点，享受便捷的交通服务。 
 
 ### 1.1 核心特性 
 - **自动搭乘**：在站台区域右击铁轨即可自动搭乘对应线路的列车。 
 - **实时信息**：站台和车厢内会通过 Title/Subtitle 实时显示当前站点、开往方向、下一站及换乘信息。 
 - **智能播报**：列车发车前有倒计时提醒，到站时会自动报站并播放提示音。 
 
 ## 章节二：乘车指南 
 > 乘车过程中请扶好站稳，注意安全. 
 
 ### 2.1 乘车流程 
 1. 前往地铁站台。 
 2. 观察屏幕上的 Title 信息，确认当前站点和列车方向。 
 3. 右击铁轨，系统会自动为您生成矿车并开始行程。 
 4. 到达目的站点后，系统会发出到站提醒，此时下车即可。 
 5. 如果是终点站，请务必下车，以免影响后续列车运行。 
 
 ::: tip 提示 
 如果您找不到地铁入口，请在主城地图中寻找绿色的“M”图标或通过快捷导航菜单前往。 
 ::: 
 
 ::: info 信息 
 地铁为全自动化运行，玩家无需任何额外操作或付费即可乘坐。 
 :::
