加入 舵星归途 StarMC 非常简单，但为了确保您能获得最稳定、最丝滑的游戏体验，我们建议您按照本指南进行操作。无论您是正版玩家还是离线玩家，这里都有详尽的连接方案。

## 第一部分：连接信息速查

在尝试进入之前，请先保存以下核心连接地址：

- **主线路地址**: `mc.star-mc.top` —— 经过 BGP 多线优化的主入口，适合绝大多数中国大陆玩家。
- **备用线路地址**: `play.star-mc.top` —— 位于不同机房的灾备入口，当主线遇到不可抗力波动时可作为备选。
- **模组专用入口**: `mod.star-mc.top` —— 用于连接特定版本的模组实验服。

## 第二部分：客户端版本与环境准备

### 1. 推荐版本
虽然我们的服务器支持跨版本连接，但由于底层核心运行在 1.21 系列，我们**强烈建议**您使用以下版本：
- **Java Edition 1.21.x**: 这是获得最完美方块渲染和光影效果的唯一选择。

### 2. 配置 Java 环境 (JDK)
这是游戏能否启动的核心。由于服务器运行在 1.21 版本，您必须安装 **Java 17 或更高版本**。

**推荐方案：安装 Java 21 (LTS)**
Java 21 是目前的长期支持版本，具有更好的性能和稳定性。
- **[官方下载地址 (Adoptium)](https://adoptium.net/zh-CN/temurin/releases/?version=21)**：进入页面后选择您的操作系统（如 Windows），下载 `.msi` 或 `.exe` 安装包。
- **[群文件快速获取]**：您也可以直接在官方 Q 群文件中搜索 `JDK 21` 下载。

**配置步骤：**
1. **执行安装**：双击下载的安装包，一路点击“下一步”直到完成。
2. **指定路径**：
   - 打开您的游戏启动器（PCL2、HMCL 等）。
   - 进入 **“设置”** -> **“游戏设置”** 或 **“Java 设置”**。
   - 在 **“Java 路径”** 一栏中，点击浏览，指向您刚安装的目录。
   - **关键文件**：通常需要选中安装目录 `bin` 文件夹下的 `javaw.exe`。
     - *默认路径参考*：`C:\Program Files\Eclipse Adoptium\jdk-21.x.x\bin\javaw.exe`

::: warning 注意事项
如果您使用的是旧版 Java 8，游戏在启动 1.21 版本时会直接崩溃或提示“不支持的 Java 版本”。
:::

## 第三部分：详细连接步骤（按启动器分类）

### 方案 A：使用 PCL2 启动器（最推荐）
1. 打开 PCL2，点击“多人游戏”。
2. 点击“添加服务器”，在“服务器地址”栏填入 `mc.star-mc.top`。
3. 如果您提示“缺少补丁”，PCL2 会自动引导您下载。
4. 点击进入，如果您是第一次进入且提示注册，请参考下文的指令说明。

### 方案 B：使用 HMCL 启动器
1. 确保已开启“版本隔离”功能。
2. 在多人游戏列表中添加我们的服务器地址。
3. HMCL 可能会提示“服务器信息加载中”，这属于正常现象，直接双击连接即可。

### 方案 C：使用正版官方启动器
1. 确保您已登录正版微软账号。
2. 直接添加服务器地址即可。StarMC 完美兼容正版验证，系统会自动识别您的正版身份，并同步您的皮肤。

## 第四部分：入服指令与账号安全

### 1. 注册与登录
- **初次注册**: 当您第一次进入服务器时，请按 `T` 键打开聊天框，**直接输入您的密码并回车**（不需要加 `/register` 前缀）。系统会要求您输入两次以确认。
- **后续登录**: 每次进服时，直接在聊天框输入您的密码即可。
- **修改密码**: 如果您想更换密码，请在登录状态下输入指令 `/changepassword 旧密码 新密码`。

::: tip 安全提示
虽然注册和登录不需要输入指令前缀，但修改密码等管理操作仍需使用标准指令格式。
:::

### 2. 关于离线角色名
- **重要提醒**: 离线角色名是区分大小写的。例如 `Steve` 和 `steve` 在服务器眼中是两个完全不同的账号。请务必记住您初次注册时的精确写法，否则进服后会发现自己的物资和领地全部消失（实际上是进错了账号）。

## 第五部分：常见连接报错排查

### 1. "Authentication Failed" (身份验证失败)
- **原因**: 您勾选了正版登录模式，但您的账号尚未在 Mojang 官网购买游戏，或者网络环境无法连接到微软验证服务器。
- **对策**: 尝试切换到“离线登录”模式。

### 2. "Timed Out" (连接超时)
- **原因**: 您的网络到服务器主入口的路径出现了拥堵。
- **对策**: 尝试使用备用线路 `play.star-mc.top`，或者重启您的家用路由器以刷新 IP 路由。

### 3. "Internal Exception: java.io.IOException"
- **原因**: 这通常是由于本地防火墙或杀毒软件拦截了 Java 的联网权限。
- **对策**: 将 javaw.exe 添加到系统的防火墙白名单中。
