## 概览表


| 分类   | 说明               | 适合物品              | 玩法倾向      |
| ---- | ---------------- | ----------------- | --------- |
| 战斗附魔 | 提升输出、压制和反制能力     | 剑、斧、三叉戟、重锤、盾牌    | PVE / PVP |
| 工具附魔 | 提升采集、种植、经验和耐久体验  | 镐、斧、锹、锄、钓鱼竿等      | 日常效率      |
| 三叉戟/特殊附魔 | 三叉戟与特殊武器的专属附魔 | 三叉戟、山羊角          | 特殊玩法     |
| 弓弩附魔 | 强化远程命中、追踪、弹射与压制  | 弓、弩              | 狩猎与远程作战   |
| 防具附魔 | 提升生存、机动、防爆、抗性和反制 | 胸甲、护腿、靴子、头盔、鞘翅、盾牌 | 稳健生存      |
| 通用附魔 | 不限武器，可跨装备通用        | 多数武器 / 工具          | 灵活搭配     |
| 诅咒附魔 | 有副作用，但通常伴随特殊收益   | 多数装备与工具           | 风险换收益     |

## 互斥规则说明


| 规则类型   | 说明                                  |
| ------ | ----------------------------------- |
| 自定义互斥  | 某些自定义附魔之间不能共存，例如同一定位的进攻、反制或移动类附魔。   |
| 原版互斥   | 部分自定义附魔会和原版附魔冲突，例如远程、火焰、效率相关组合。     |
| 实际搭配原则 | 先保留当前玩法最有用的那一本，再补其他位。不要把同定位附魔硬塞到一起。 |
