# 加入教程

这页只讲一件事：怎么顺利进服。你不需要懂技术，只要按顺序做，就能比较稳地进入舵星归途 StarMC。

> **先看结论**
> 推荐使用 Java 版 1.21.x，默认先连 `mc.star-mc.top`。如果进不去，再换其他线路。

## 一分钟快速加入

如果你想马上开始玩，先按下面的顺序来：

1. 使用 **Java 版 1.21.x**
2. 打开多人游戏，添加服务器
3. 地址填 `mc.star-mc.top`
4. 进入服务器后允许资源包
5. 按聊天栏提示完成登录或注册

先不用纠结太多细节。能正常进入，再慢慢处理其他设置。

## 服务器地址怎么选

| 线路 | 地址 | 建议用途 |
| :--- | :--- | :--- |
| 主线路 | `mc.star-mc.top` | 大多数玩家优先使用 |
| 主力线路 | `max.star-mc.top` | 主线路卡顿、超时或延迟高时尝试 |
| 备用线路 | `star-mc.top` | 前两条线路不稳定时再试 |
| 模组实验入口 | `mod.star-mc.top` | 仅用于对应实验内容 |

### 线路选择建议

- 先连主线路，不要一开始就换来换去。
- 如果提示超时，换一条线通常比反复重进更有效。
- 如果你正在用加速器或代理，先确认它没有干扰游戏连接。

## 推荐版本

推荐使用：

- **Java 版 1.21.x**

使用更旧版本时，常见问题有：

- 进不去服务器
- 资源包加载不完整
- 方块、字体或物品显示异常
- 部分功能不能正常使用

如果你不确定选哪个版本，直接用推荐版本就好。

## 启动器和 Java

如果你用的是 Java 版启动器，建议确认 Java 版本：

- 至少 Java 17
- 更推荐 Java 21

### 什么时候要先查 Java

如果你遇到这些问题，先别急着重装游戏：

- 启动就闪退
- 黑屏
- 提示 Java 错误
- 游戏一直卡在启动界面

PCL2、HMCL、官方启动器一般都能手动选择 Java 路径。如果你不会设置，可以直接截个图发给熟悉的玩家或管理。

## 添加服务器

### 常见启动器

1. 打开游戏
2. 进入“多人游戏”
3. 点击“添加服务器”
4. 服务器地址填写 `mc.star-mc.top`
5. 保存并进入

### 官方启动器

1. 使用微软账号登录
2. 选择推荐版本启动游戏
3. 进入“多人游戏”
4. 添加服务器地址 `mc.star-mc.top`
5. 进入服务器

## 第一次进服会发生什么

第一次进入时，聊天栏可能会提示你注册、登录或确认身份。请直接按提示操作。

### 记住这几件事

- 角色名大小写要记清楚
- 不要把密码发到公屏
- 密码不要太简单
- 忘记密码就按公告指引联系管理

## 资源包要开吗

建议开启。否则你可能会看到：

- 材质错乱
- 图标不完整
- 文字显示异常
- 自定义内容没有正常加载

如果你在服务器列表里能设置资源包，建议选：

- **启用**
- 或 **提示**

更多细节可以看 [资源加载](./core/ResourceLoading)。

## 进不去时先怎么排查

### 提示连接超时
按这个顺序换线路：

1. `mc.star-mc.top`
2. `max.star-mc.top`
3. `star-mc.top`

### 提示版本不匹配
换成 Java 版 1.21.x。

### 启动就闪退
检查 Java 版本，优先用 Java 17 或 Java 21。

### 能进服但画面怪
大概率是资源包没有正确加载。先看 [资源加载](./core/ResourceLoading)。

### 一直掉线或网络异常
检查本地网络、防火墙、安全软件、代理或加速器设置。

## 还是不行怎么办

联系管理时，尽量一次性说清楚这些信息：

- 你的游戏 ID
- 游戏版本
- 启动器名称
- 正在使用的服务器地址
- 报错截图或报错文字
- 你已经试过哪些方法

这样比只说“我进不去”更容易得到有效帮助。
