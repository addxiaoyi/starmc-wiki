# 跨平台连接指南

服务器支持 Java 版（电脑）与基岩版（手机、平板、Win10/11 版）玩家同服游戏。我们针对基岩版做了专门的 UI 适配，操作非常方便。

## 🎮 基岩版专属 UI
为了让手机玩家不用背繁琐的指令，我们把大部分功能都做成了原生的表单界面（GUI）。

### 快捷指令表
如果你是基岩版玩家，输入以下指令可以直接弹出操作窗口：

| 指令 | 功能 |
| :--- | :--- |
| `/homegui` | 管理家园、传送回家 |
| `/warpgui` | 打开公共地标传送菜单 |
| `/tpgui` | 发起或接收玩家传送请求 |
| `/msggui` | 快速发送私聊消息 |
| `/kitgui` | 领取新手礼包或各类奖励 |
| `/phomegui` | 浏览并前往其他玩家的公开家园 |

> **死亡提醒**
> 基岩版玩家死亡复活后，系统会自动弹出返回死亡点的表单，点一下就能回去。

---

## 🔗 连接与同步

### 1. 账号关联
- **自动创建**：基岩版进服会自动关联一个 Java 版身份，不需要额外注册。
- **材质同步**：进服后会自动加载服务器资源包，确保你能看到自定义的物品和方块。

### 2. 平台兼容建议
| 平台 | 连接建议 | 备注 |
| :--- | :--- | :--- |
| **Android / iOS** | 直接搜索服务器 IP | 完美支持 UI 表单 |
| **Win10 / 11 版** | 在“服务器”列表添加 | 体验与 Java 版最接近 |
| **主机 (Xbox/PS)** | 通过 DNS 转发连接 | 需要简单的网络配置 |

::: tip 小提示
如果你在手机上觉得菜单太难点，记得直接用上面的 `/gui` 类指令，弹出式窗口操作更顺手。
:::
