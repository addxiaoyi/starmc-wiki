# 跨平台连接指南

服务器支持 Java 版和基岩版玩家同服。如果你是手机、平板或 Win10/11 基岩版玩家，这页会告诉你怎么更顺利地进服和使用功能。

## 先看结论

基岩版玩家一般可以直接正常进服，不需要把自己当成“特殊版本”。大多数功能都已经尽量做成好用、直观的形式了。

## 基岩版玩家能做什么

为了让基岩版玩家更容易上手，服务器准备了很多更直观的操作方式，比如：

- 回家传送
- 地标传送
- 玩家传送
- 私聊
- 领取奖励
- 浏览公开家园

如果你不想记很多指令，可以优先使用这些更方便的入口。

## 常见的便捷指令

| 指令 | 功能 |
| :--- | :--- |
| `/homegui` | 管理家园、传送回家 |
| `/warpgui` | 打开公共地标传送菜单 |
| `/tpgui` | 发起或接收玩家传送请求 |
| `/msggui` | 快速发送私聊消息 |
| `/kitgui` | 领取新手礼包或奖励 |
| `/phomegui` | 浏览并前往其他玩家的公开家园 |

如果你习惯手机操作，这类界面通常会比手打指令更顺手。

## 账号和同步

### 进服后会发生什么

基岩版玩家进入服务器后，一般会自动获得对应身份，不需要额外折腾太多注册步骤。

### 材质加载

和 Java 版一样，基岩版也需要资源包来保证自定义内容正常显示。遇到材质异常时，优先先看 [资源加载](./ResourceLoading)。

## 不同平台怎么连更顺手

| 平台 | 建议 |
| :--- | :--- |
| Android / iOS | 直接搜索服务器 IP |
| Win10 / 11 基岩版 | 在服务器列表里添加 |
| 主机端 | 通常需要额外的网络配置 |

## 你如果是手机玩家

手机玩家最常见的问题不是“玩不了”，而是“点起来有点累”。这时建议：

- 多用表单界面
- 少背指令
- 常用功能固定放在自己顺手的入口
- 传送、私聊、回家都尽量用快捷方式

## 你如果是 Java 玩家

Java 玩家一般可以直接使用常规指令和菜单。跨平台不是限制，而是让更多人能一起玩。

## 如果连接不顺

优先检查：

1. 版本是否正确
2. 网络是否稳定
3. 是否加载了正确资源包
4. 是否被本地设置或代理影响

## 最后提醒

跨平台的核心不是“规则更多”，而是“让不同设备的人都能一起舒服地玩”。如果你发现某些功能不顺手，先试试更适合你设备的操作方式。
