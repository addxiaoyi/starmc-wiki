# 基础指令

这页只整理玩家最常用的指令。你不需要背完所有内容，遇到需要时回来查就行。

> **怎么看指令**
> 方括号 `[]` 表示需要你自己填写内容，实际输入时不要带括号。

例如：

```text
/sethome base
```

意思是把当前位置设置成名为 `base` 的家。

## 新玩家最常用的 5 个指令

| 指令 | 什么时候用 | 说明 |
| :--- | :--- | :--- |
| `/spawn` | 想回主城时 | 返回主城或出生点 |
| `/rtp` | 想找野外建家时 | 随机传送到野外区域 |
| `/sethome base` | 找到家以后 | 把当前位置设置成家 |
| `/home base` | 想回家时 | 回到名为 `base` 的家 |
| `/tpa 玩家名` | 找朋友时 | 请求传送到朋友身边 |

如果你刚进服，建议先会这几个就够了。

## 回家与传送

| 指令 | 用途 | 例子 |
| :--- | :--- | :--- |
| `/spawn` | 回到主城或出生点 | `/spawn` |
| `/sethome [名称]` | 设置一个家 | `/sethome base` |
| `/home [名称]` | 回到指定的家 | `/home base` |
| `/delhome [名称]` | 删除一个家 | `/delhome oldhome` |
| `/rtp` | 随机传送到野外 | `/rtp` |

### 使用建议

- 建家后第一件事：先设置家。
- 外出探索前，确认自己能回家。
- 随机传送适合找新地形，但请带好食物、木头和基础装备。
- 删除家的时候确认名字，避免误删常用传送点。

## 玩家之间传送

| 指令 | 用途 |
| :--- | :--- |
| `/tpa [玩家名]` | 请求传送到对方身边 |
| `/tpahere [玩家名]` | 请求对方传送到你身边 |
| `/tpaccept` | 接受传送请求 |
| `/tpdeny` | 拒绝传送请求 |

### 安全提醒

不要随便接受陌生人的传送请求。交易、参观、合作前，最好先确认对方身份和目的。重要基地不要轻易带陌生人进入。

## 聊天与信息

| 指令 | 用途 |
| :--- | :--- |
| `/msg [玩家名] [内容]` | 私聊某位玩家 |
| `/r [内容]` | 快速回复最近私聊你的人 |
| `/list` | 查看当前在线玩家 |
| `/ping` | 查看你的延迟 |

### 小技巧

- 私聊适合交易、求助和坐标沟通。
- 如果延迟明显偏高，可以先换线路，或者稍后再尝试连接。
- 公屏聊天请保持基本礼貌，不要刷屏或恶意挑衅。

## 经济与交易

| 指令 | 用途 |
| :--- | :--- |
| `/money` | 查看自己的余额 |
| `/bal` | 查看自己的余额，和 `/money` 类似 |
| `/pay [玩家名] [金额]` | 给其他玩家转账 |

### 交易提醒

- 大额交易建议保留聊天记录。
- 不要先付款给不熟悉的玩家，除非你能接受风险。
- 涉及稀有物品、装备、领地时，建议找可信玩家或管理见证。
- 转账前确认玩家名，避免把钱转给同名相似的账号。

## 应急指令

| 指令 | 用途 |
| :--- | :--- |
| `/suicide` | 自杀，适合严重卡死且无法脱困时使用 |

这个指令会造成死亡后果。能通过传送、求助、重进解决的问题，不建议直接使用。

## 输入技巧

- 多用 `Tab` 自动补全玩家名和指令。
- 不确定指令怎么写时，可以先输入前半段再按 `Tab`。
- 指令一般不区分大小写，但玩家名可能区分大小写。
- 如果提示“没有权限”或“未知指令”，先确认是否输错，再去 FAQ 或官方群询问。

## 推荐记法

不用背完整指令表，只记三个场景：

1. **回家**：`/sethome`、`/home`
2. **找人**：`/tpa`、`/tpaccept`
3. **交易**：`/money`、`/pay`

其他指令需要时再回来查即可。
