# 常见问题 FAQ

这里整理的是玩家最常遇到的问题。遇到问题时，建议先按分类快速看一遍，很多情况不用等管理也能自己解决。

> **排查顺序建议**
> 先确认版本和线路，再看资源包；如果涉及账号、封禁、纠纷，再联系管理处理。

## 入服与账号

### 我是正版玩家，为什么还提示注册？
这通常是因为你的角色名曾经被离线账号注册过。请联系管理，并提供能够证明正版身份的截图或信息。核实后，管理会协助你处理旧账号占用问题。

### 忘记密码怎么办？
请通过服务器公告中指定的渠道联系管理。准备好你的游戏 ID，并按要求完成身份核实。为了账号安全，不要在公屏发送密码、旧密码或类似信息。

### 离线账号为什么要注意大小写？
服务器可能会把大小写不同的名字视为不同账号。例如 `Steve` 和 `steve` 可能不是同一个角色。第一次注册后，请记住你的准确角色名写法。

### 改了名字还能找回原来的东西吗？
不一定。角色数据通常和游戏名绑定，改名可能影响背包、领地、经济或权限识别。改名前建议先问管理，确认服务器当前的处理方式。

## 连接与版本

### 为什么我进服超时或连接不上？
先按这个顺序排查：

1. 优先使用 `mc.star-mc.top`
2. 不行再试 `max.star-mc.top`
3. 还不行再试 `star-mc.top`
4. 检查本地网络、防火墙、代理或加速器设置

如果某一条线路临时不稳定，换线路通常比反复重进更有效。

### 推荐使用什么版本？
推荐使用 **Java 版 1.21.x**。服务器可能支持一定范围内的跨版本连接，但推荐版本通常最稳定，资源包、物品显示和玩法兼容性也最好。

### 游戏启动就闪退怎么办？
优先检查 Java 版本。1.21 系列通常建议使用 Java 17 或 Java 21。如果启动器仍在使用 Java 8，很容易出现闪退、黑屏或无法启动。

### 为什么别人能进，我进不去？
常见原因包括：本地网络不稳定、版本不匹配、Java 版本过旧、资源包异常、防火墙拦截。建议先按 [加入教程](./join) 的排查顺序处理。

## 资源包与显示问题

### 材质、文字或图标显示不正常怎么办？
大多数情况是服务器资源包没有正确加载。请先到 [资源加载](./core/ResourceLoading) 检查设置，确认多人游戏里的服务器资源包已经设为“启用”或“提示”。

### 为什么会看到奇怪方块或“假矿”？
服务器可能启用了反矿透保护。区块刚加载、网络波动或反作弊干预时，可能短暂看到异常方块。通常重新靠近、重新加载区块或稍等一会儿就会恢复。

### 资源包下载很慢怎么办？
可以先重进服务器，或换一条连接线路再试。如果一直下载失败，截图发给管理，说明你使用的版本、启动器和报错信息。

## 建筑、领地与物资

### 我的建筑被拆了，能查到是谁吗？
服务器有日志记录，通常可以调查方块和容器操作。请尽量提供时间、坐标、截图或视频。信息越完整，处理越快。

### 没圈地的建筑被破坏了怎么办？
可以找管理尝试调查，但未圈地区域的保护能力有限，恢复也不一定完整。建家后请尽快阅读 [领地系统](./residence)，先把重要区域保护起来。

### 服务器会删档吗？
正常情况下不会。除非遇到极少数不可抗力或重大架构调整，服务器会尽量保持长期存档稳定。重大调整会尽量提前公告。

### 可以和朋友共用一个基地吗？
可以。建议基地所有者先圈地，再通过领地权限给朋友开放建造、箱子、使用等权限。不要直接把所有东西放在无保护区域。

## 玩法与规则

### 大型红石机器可以建吗？
可以，但要控制规模和运行方式。机器最好有开关，不要长期高频运行，也不要明显影响全服性能。详细要求请看 [红石限制说明](./redstone)。

### 自动农场、刷怪塔可以用吗？
可以正常建设，但不能利用漏洞、不能无人值守刷到影响服务器性能，也不能恶意堆实体或掉落物。发现机器卡服时，请配合管理优化。

### 发现漏洞怎么办？
不要传播，不要利用，尽快联系管理并说明触发方式。主动上报严重漏洞通常会被正向处理；私下利用漏洞获利会按作弊处理。

## 封禁与申诉

### 我被封禁了，去哪里申诉？
请通过服务器公告指定的渠道联系管理，并准备：

- 游戏 ID
- 封禁提示截图
- 事情经过说明
- 相关聊天记录、截图或录像

### 怎么更容易申诉成功？
把事实说清楚，比情绪化争论更有用。时间、地点、涉及玩家、操作经过越完整，管理越容易判断。

### 被误判怎么办？
先保留报错、封禁提示或相关截图，再联系管理。不要刷屏、不要用小号反复进服，这会让问题更难处理。

## 社区相关

### 遇到问题去哪里问最快？
建议按顺序处理：

1. 先查 Wiki
2. 再看 FAQ
3. 仍然解决不了，再去官方群找管理或技术组

### 怎么成为管理？
一般不会直接招“空降管理”。长期活跃、愿意帮助新玩家、熟悉服务器玩法、处理事情成熟可靠的玩家，更有机会被考虑。

### 可以宣传自己的群或服务器吗？
不建议在公屏或私聊里拉人跳服、发广告或刷屏宣传。正常交流可以，影响社区秩序的推广会被处理。

## 还没解决怎么办

联系管理时，建议一次性说明：

- 你的游戏 ID
- 发生时间
- 所在坐标或服务器区域
- 你已经尝试过的解决办法
- 截图、视频或报错内容

这样比只说“我进不去”“我东西没了”更容易得到有效帮助。
