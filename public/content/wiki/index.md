# 🌟 StarMC 百科

StarMC 官方百科，涵盖从入服指南到进阶机制的所有内容

> **新手建议**
> 刚来的玩家建议先看 [加入教程](./join) 和 [服务器规范](./rules)

---

## 🛠️ 接入与资源
确保游戏显示正常和连接顺畅的基础设置

| 模块 | 功能 | 说明 |
| :--- | :--- | :--- |
| **[资源加载](./core/ResourceLoading)** | 解决方块字/材质 | 记得开启服务器资源包 |
| **[跨平台连接](./core/CrossPlatform)** | Java/基岩版互通 | 基岩版端口 19132 |
| **[世界边界](./core/WorldLimits)** | 领地/高度限制 | 建设上限可达 1024 格 |

## 🌾 玩法与机制
在原版生存的基础上，我们加了些让游戏更耐玩的机制

### 1. 生存增强
- **[农作物种植](./gameplay/Farming)**：23 种新作物，包含季节枯萎机制
- **[趣味垂钓](./gameplay/Fishing)**：重新设计的掉落池，能钓到不少好东西
- **[技能系统](./gameplay/Skills)**：11 项可升级技能，慢慢积累你的优势

### 2. 社区与社交
- **[个性化称号](./social/Identity)**：UI 界面管理你的成就头衔
- **[婚姻系统](./social/Marriage)**：支持双人传送和特殊互动
- **[交互增强](./social/Interaction)**：支持坐下、躺下、骑乘等动作
- **[快捷展示](./social/ChatEnhance)**：聊天框输入 `[item]` 秀出你的装备

## 💰 经济与便利
- **[贸易商店](./utility/Trading)**：玩家自建商店，低门槛自由贸易
- **[每日任务](./utility/Quests)**：每天更新，涵盖收集、击杀和探索
- **[地标传送](./utility/Warp)**：方便在主要区域快速穿梭
- **[签到福利](./utility/CheckIn)**：累积签到领点基础物资
