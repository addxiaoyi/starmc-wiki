# 附魔冲突说明

有些附魔不能同时装在同一件物品上。可以理解成：这些附魔走的是不同路线，硬装在一起反而会互相顶掉。

<div class="page-hero-card">

**快速看点**

| 项目 | 内容 |
| --- | --- |
| 收录范围 | 当前默认冲突策略下的全部冲突关系 |
| 冲突方向 | 双向冲突，A 冲突 B 等于 B 冲突 A |
| 内容类型 | 自定义附魔冲突 + 原版附魔冲突 |

</div>

## 先记住一个原则

| 原则 | 说明 |
| --- | --- |
| 不要全装 | 不是能装就一定要一起装。 |
| 先选路线 | 很多附魔更适合“路线选择”，不是“全部叠满”。 |
| 先看用途 | 先决定你要什么风格，再去挑附魔。 |

## 常见冲突怎么理解

| 分类 | 常见冲突来源 | 直观理解 |
| --- | --- | --- |
| 盾牌类 | 更稳 / 更能反打 / 更像功能辅助 | 这些方向在做不同的事，常常互相冲突。 |
| 弓弩类 | 多箭 / 单体爆发 / 稳定输出 / 特殊弹道 | 一把武器通常更适合其中一种主思路。 |
| 火焰类 | 冰火路线 | 通常二选一，先选控场还是持续灼烧。 |
| 近战收割类 | 补刀 / 爆发 / 持续压制 | 都在争同一个位置，往往不能混装。 |

## 自定义附魔冲突

| 分类 | 相关附魔 | 说明 | 触发方法 |
| --- | --- | --- | --- |
| 盾牌相关 | `守护`、`固握`、`稳架`、`转守`、`借势`、`回击`、`壁垒` | 大多围绕盾牌的不同防守路线，通常不能乱叠。 | 盾牌格挡、受击、反击时触发 |
| 飞行相关 | `安全气囊`、`羽护`、`缓翼` | 都和鞘翅生存有关，但思路不同。 | 受伤、滑翔、坠落时触发 |
| 弓弩相关 | `齐射`、`远袭`、`定准`、`静止`、`弹返`、`猎印` | 要在清怪、单点、稳定和特殊弹道里做选择。 | 发射、命中、稳定出手时触发 |
| 近战相关 | `处刑者`、`斩首`、`血蚀`、`影袭`、`先机` | 更适合按补刀、爆发、持续输出来分。 | 攻击命中、击杀、目标状态变化时触发 |
| 火焰相关 | `焦灼`、`霜刃`、`火焰附加` | 冰火路线不要硬叠。 | 近战命中时触发 |

## 原版附魔冲突

| 原版附魔 | 常见对应冲突 | 触发方法 | 说明 |
| --- | --- | --- | --- |
| `多重射击` | 某些弓弩附魔 | 发射远程武器时触发 | 多箭、爆发和特殊弹道通常不能混得太满。 |
| `锋利` | 某些近战附魔 | 近战攻击命中时触发 | 近战收割型附魔常常和锋利路线有冲突。 |
| `精准采集` | 某些采集附魔 | 破坏方块时触发 | 采集路线需要先判断是“掉落收益”还是“原样获取”。 |

## 怎么判断该不该换掉

| 步骤 | 要问自己什么 | 触发方法 |
| --- | --- | --- |
| 1 | 这件装备主要干什么？ | 先看你平时怎么用它 |
| 2 | 我最常用的场景是什么？ | 在常用场景里是否触发价值 |
| 3 | 哪个附魔更符合这个场景？ | 哪个更常触发、收益更大 |
| 4 | 被冲掉的那个是不是只是看起来很强，但其实不常用？ | 是否真的值得保留 |

## 新手最实用的建议

| 建议 | 说明 | 触发方法 |
| --- | --- | --- |
| 先做基础附魔 | 再考虑冲突优化。 | 先把常用效果做起来 |
| 不要全收 | 别为了全收而牺牲常用功能。 | 看到更强不代表更适合 |
| 先定路线 | 先决定路线，再决定附魔。 | 先看自己的玩法定位 |
| 优先保留常用项 | 真纠结时先留更常用、更稳的那一个。 | 常触发、常生效的优先 |

## 什么时候一定要看这页

| 场景 | 为什么要看 | 触发方法 |
| --- | --- | --- |
| 配毕业武器 | 避免把关键输出冲掉。 | 准备成型装备时触发判断 |
| 做盾牌流 | 盾牌路线冲突特别多。 | 盾牌格挡和反击前先看 |
| 选弓弩主输出 | 远程路线常常需要二选一。 | 远程发射前先确认 |
| 给鞘翅加保护 | 飞行类附魔组合有限。 | 滑翔、坠落、受伤时先判断 |
| 做“最顺手”的工具 | 工具路线也可能互相冲突。 | 挖矿、采集、掉落处理前先看 |
