# 附魔获取途径

这页专门解释附魔表里常见的获取字段。你可以把它当成“这本附魔从哪来”的速查说明。

<div class="page-hero-card">

**快速看点**

- 看懂“普通 / 特殊 / 宝藏 / 纯宝藏”分别是什么意思
- 先判断这本附魔是更容易刷，还是更偏稀有
- 适合在附魔前先做一次来源确认

</div>

## 你先记这几个词

### 附魔台普通

最常见、最容易碰到的来源。适合新手优先收集，也适合日常补基础附魔。

### 附魔台特殊

还是能从附魔台出，但通常更看运气，也更偏功能型或强力型。

### 宝藏特殊

更偏稀有奖励，通常不会像普通附魔那样顺手刷到。适合后期慢慢补。

### 纯宝藏

基本不是你随手附魔台就能出的类型。更适合把它当作长期目标。

### 诅咒池

单独的负面附魔来源。想玩诅咒或整蛊路线时才会重点关注。

### 村民交易

更稳定、适合定向收集。想慢慢补齐某些附魔时特别有用。

### 战利品

从箱子、掉落池、宝箱等奖励里拿到。适合跑图、开箱、刷地牢时顺带收集。

## 怎么理解这些来源

| 来源 | 直观理解 | 对玩家意味着什么 |
| :--- | :--- | :--- |
| 附魔台普通 | 最容易刷到 | 适合前期和常用附魔 |
| 附魔台特殊 | 能刷，但看运气 | 适合功能型或较强附魔 |
| 宝藏特殊 | 比普通更稀有 | 适合后期定向收集 |
| 纯宝藏 | 很难顺手刷到 | 适合作为长期目标 |
| 诅咒池 | 负面附魔专用 | 只在想玩诅咒时关注 |
| 村民交易 | 更稳定 | 适合定向补书 |
| 战利品 | 开箱和探索获取 | 适合跑图时顺手收集 |

## 怎么用这页最省时间

如果你在附魔总表里看到某本附魔的获取方式，先问自己三个问题：

1. **我现在能不能稳定拿到？**
2. **我要不要专门去刷？**
3. **它是不是更适合后期再收？**

这样你就不用看到“宝藏”“特殊”就发懵。

## 新手怎么选

- **前期**：优先找普通来源
- **中期**：开始补特殊来源和村民交易
- **后期**：再去收宝藏和纯宝藏

## 最常见的误解

### 误解 1：越稀有就越适合我

不一定。很多稀有附魔只是更难拿，不代表对你当前玩法最有帮助。

### 误解 2：只有最强附魔才值得收

不是。能帮你更稳、更顺手、少死几次的附魔，往往比“传说级但用不上”的更实用。

### 误解 3：来源越复杂越高级

来源复杂通常只代表更难拿，不一定代表更适合所有人。
