# 领地保护

纯生存模式下，领地系统（Residence）是保护建筑和物资的手段。没有领地的区域任何人都能修改。

## 创建领地

### 1. 准备工具
您需要一把**木锄（Wooden Hoe）**作为圈地工具。

### 2. 选定范围
领地是一个长方体区域。您需要选定对角线的两个端点：
- **点 A**: 对准第一个角，**左键**点击方块。
- **点 B**: 对准对角线的另一个角，**右键**点击方块。
- 系统会提示您已选定两个点。

### 3. 创建指令
输入 `/res create [领地名称]` 即可创建。
- **注意**: 领地创建需要支付一定的星币（按方块体积计费）。如果余额不足，请参考经济系统获取星币。

## 领地权限管理

一旦领地创建成功，默认只有您可以进行破坏和交互。您可以根据需要给好友授权。

### 1. 赋予权限
- `/res pset [领地名] [玩家名] [权限名] true`: 开启某项权限。
- 常用权限名：
    - `build`: 放置和破坏方块。
    - `container`: 打开箱子、熔炉等。
    - `use`: 使用按钮、拉杆、门。
    - `tp`: 允许该玩家传送到您的领地。

### 2. 移除权限
- `/res pset [领地名] [玩家名] [权限名] remove`: 将该权限恢复为默认（禁止）。

## 子领地
如果您想将领地的一部分租给他人，或者为不同房间设置不同权限：
- `/res subzone [主领地名] [子领地名]`: 在已选定范围的基础上创建子领地。

## 常用指令
- `/res tp [领地名]`: 传送到您的领地。
- `/res set`: 打开图形化权限设置界面（最推荐，简单直观）。
- `/res info`: 查看当前领地的详细信息（所有者、权限列表、价格等）。
- `/res remove [领地名]`: 删除领地。删除后将返还部分资金，但区域将立即失去保护。

**注意**: 建好房子后尽快圈地，未圈地导致的损失管理团队不负责赔偿。
