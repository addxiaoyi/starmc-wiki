# 世界边界与高度

服务器突破了原版 Minecraft 的限制，为你提供了更广阔的建设和挖掘空间。

## 🚀 扩展特性
- **建设高度**：最高可建设至 **1024 层**（具体视区域而定），让你能造出真正的摩天大楼。
- **挖掘深度**：最深可挖掘至更深的负层，发现更多矿藏。
- **区块加载**：经过底层优化，即使在高处建筑也能保持流畅显示。

## 💡 注意事项
- **视距建议**：如果你在高处建筑，建议将视距调至 **8 以上**，否则可能看不到地面的景观。
- **坠落风险**：虽然高度增加了，但坠落伤害依然在，在高处干活记得带上鞘翅或缓降药水。
- **自动适配**：这是一个底层功能，不需要输入指令，直接往上盖或往下挖就行。

::: tip 小提示
极高处的建筑可能会因为云层遮挡影响视野，你可以尝试在设置里关闭云层显示。
:::
