# 自定义资源加载

为了让服务器里的家具、作物、称号和 UI 图标正常显示，我们需要依赖资源包。

## ⚙️ 核心机制
服务器会自动合并多个插件的材质（如农作物、称号、模型等），并生成统一的资源包发给客户端。

- **版本要求**：推荐使用 Minecraft **1.21.1** 及以上版本。
- **自动应用**：进服时系统会提示你下载资源包，点击“是”即可。
- **性能优化**：我们剔除了资源包里的冗余文件，并针对高版本做了性能优化，不用担心拖慢游戏。

## 🛠️ 常见问题处理

### 1. 材质丢失（紫黑方块）
如果看到物品变成了紫黑方块，通常是资源包没加载成功。
- **解决方法**：尝试退出并重新进入服务器。
- **手动获取**：输入 `/ce pack` 获取资源包下载链接，手动安装。

### 2. 聊天框或 UI 出现方块字
这是因为自定义图标的字体与你本地的材质包冲突了。
- **建议**：暂时关闭本地带有自定义字体的材质包，优先使用服务器提供的资源。

## ⌨️ 常用指令
| 指令 | 功能 |
| :--- | :--- |
| `/ce pack` | 获取当前资源包的下载链接 |
| `/ce reload` | (管理员) 强制重新加载资源引擎 |
