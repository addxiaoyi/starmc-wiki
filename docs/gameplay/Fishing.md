# 趣味垂钓系统

服务器对原版钓鱼做了大改，现在钓鱼不仅仅是挂机，还有更多惊喜和挑战。

## 🐠 你能钓到什么
除了原版的鳕鱼和鲑鱼，你现在有机会钓到：
- **特色鱼类**：带有特殊材质的 **彩虹鱼**、具有收藏价值的 **金枪鱼**（分星级品质）等。
- **珍稀宝藏**：包括**经验修补**、**锋利 IV** 等 40 多种高级附魔书，还有钻石、金锭等财宝。
- **奇特杂物**：运气不好也可能钓到“无用的木棍”或者“有股臭味的旧鞋子”。

## 🎮 核心玩法
- **垂钓小游戏**：钓稀有鱼时会触发小游戏（比如保持平衡或快速连点），操作成功才能钓上来。
- **成就系统**：每条鱼都有重量和长度记录，通过 `/fishing log` 查看自己的垂钓图鉴。
- **经验修复**：成功钓鱼有概率触发装备的经验修补效果，回 1~7 点耐久。
- **经济收益**：稀有鱼可以直接卖成金币（比如彩虹鱼一条能卖 15 金币）。

## ⌨️ 常用指令
| 指令 | 功能 |
| :--- | :--- |
| `/fishing log` | 查看已解锁的鱼类图鉴和垂钓记录 |
| `/customfishing stats` | 查看你的钓鱼等级和当前属性 |

::: tip 小提示
钓不到好东西？换个水域试试，或者等下雨天，运气会好很多。
:::
