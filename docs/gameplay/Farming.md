# 农作物种植指南

服务器加入了 23 种特色作物，从大蒜、卷心菜到菠萝、火龙果，带给你更真实的农场经营体验。

## 🍅 作物种类
除了原版作物，你还可以种植以下特色农产：
- **蔬菜类**：大蒜、洋葱、卷心菜、生菜、大葱、黄瓜、茄子、辣椒、甜薯等。
- **水果与经济作物**：苹果、玉米、菠萝、火龙果、葡萄、青豆、啤酒花等。
- **特殊植物**：发光真菌、维瑞莉亚，甚至还有能收获奖励的“红包”作物。

## 💧 种植要点
特色作物的生长需要考虑水分、肥料和季节。

### 1. 水分管理
- **手动浇水**：使用浇水壶（Watering Can）手动灌溉。
- **自动灌溉**：制作洒水器（Sprinkler）可以自动为周围土地提供水分，解放双手。

### 2. 肥料与生长
- **肥料**：使用特定的肥料可以加速生长、增加产量或提升作物质量。
- **季节机制**：注意！部分作物只能在特定季节生长，非季节作物可能会停止生长甚至枯萎。

## 🛠️ 常用农具
| 物品 | 功能 |
| :--- | :--- |
| **洒水器** | 放在田间，自动湿润周围耕地 |
| **肥料** | 提升产量、加速生长或改善品质 |
| **浇水壶** | 基础灌溉工具，可升级以覆盖更多格子 |

::: tip 小提示
如果作物不长了，先看看土是不是干了，或者现在的季节对不对。
:::
