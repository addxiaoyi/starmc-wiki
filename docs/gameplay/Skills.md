# 个人能力与属性

在 StarMC 里的各种活动（挖矿、种地、打怪等）都会提升你的技能等级，让你变得更强。

## 🧬 核心技能 (共 11 项)
每项技能最高 **100 级**，升级能永久提升你的基础属性：

- **采集类**：农业、采伐、采矿、钓鱼、挖掘。提升生命值、力量和双倍掉落概率。
- **战斗类**：战斗、弓箭、防御。提升伤害、暴击和减伤。
- **功能类**：敏捷、炼金、附魔。提升移速、药水时长和经验获取。

## ⚡ 技能收益
- **基础属性**：每升一级都会加点血量、力量或运气。
- **被动能力**：自动解锁。比如采矿时概率掉双倍，砍树时掉落更多原木。
- **主动技能**：手持工具**右键点击**即可准备，随后进行挖掘/采集即可触发大范围收割或超快加速。

## ⌨️ 常用指令
| 指令 | 功能 |
| :--- | :--- |
| `/skills` | 打开主菜单，查看等级、属性和已解锁的技能 |
| `/skills top` | 查看全服技能排行榜 |

::: tip 小提示
想快速升级？专注做一件事（比如专门去挖矿或种地）是最高效的。记得多看 `/skills` 里的进度。
:::
