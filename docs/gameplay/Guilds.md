 # 公会与团队建设 
 
 建立属于您的公会，与志同道合的伙伴共同成长、建设、战斗，共享公会福利。 
 
 ## 章节一：基础介绍 
 创建公会需要一定的资金投入，公会成员可以通过捐赠来提升公会等级。 
 
 ### 1.1 核心机制
- **创建公会**：
  - **基础费用**：需支付 **20,000 金币** 及 **1,000 点券**。
  - **名称限制**：名称长度上限为 **12个字符**（含颜色代码）。
  - **冷却时间**：退出、解散或被踢出公会后，需等待 **240分钟 (4小时)** 才能再次加入或创建公会。
- **公会成长**：
  - **等级上限**：公会最高可提升至 **5级**。
  - **成员上限**：随等级提升，成员容量从 **5人** 逐步扩展至 **50人**。
  - **公会贡献**：每捐赠 1.0 金币可获得 **0.1点** 贡献值。
- **公会驻地**：
  - **快捷传送**：公会成员可传送至公会驻地，传送需原地等待 **5秒**。
- **特色系统**：
  - **公会商店**：使用贡献值兑换稀有道具或公会专属 Buff。
  - **公会神树**：全员参与培育神树，解锁强力团队属性。
  - **公会红包**：成员间可派发金币或点券红包，共享喜悦。
- **管理指令**：
  - `/guild`：打开公会主菜单。
  - `/guild home`：传送至公会驻地。
  - `/guild chat`：开启公会专属聊天频道。
