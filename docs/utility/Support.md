# 服务反馈中心

无论是遇到 BUG、举报违规还是提出建议，你都可以通过反馈中心直接向管理组提交，并随时追踪处理进度。

## 🎫 提交工单
- **创建工单**：输入 `/ticket create <内容>` 描述你遇到的问题。
- **添加补充**：输入 `/ticket comment <工单ID> <内容>` 为已有的工单添加更多细节。
- **自动定位**：提交时系统会自动记录你的坐标，方便管理员核查。

## ⚙️ 使用规则
- **交互冷却**：为防止刷屏，每次操作后会有 10 秒的冷却间隔。
- **进度追踪**：输入 `/ticket` 打开主界面，即可查看你所有工单的当前状态和回复。

## ⌨️ 常用指令
| 指令 | 功能 |
| :--- | :--- |
| `/ticket` | 打开服务反馈中心界面 |
| `/ticket open <内容>` | 快速提交一个新反馈 |

::: warning ⚠️ 警告
恶意提交大量无效工单或在工单中使用不当言论，将可能导致你被永久撤销反馈权限。
:::
