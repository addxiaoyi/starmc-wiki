# 福利抽奖系统

StarMC 的彩票抽奖系统每小时开奖一次，只要参与，你就有机会清空奖池，赢取海量金币和经验。

## 🎫 参与规则
- **彩票售价**：每张 2,000.0 金币。
- **购买上限**：每人每轮最多购买 100 张。
- **开奖周期**：每 60 分钟自动开启一轮抽奖。

## 💰 奖池逻辑
- **奖金来源**：奖池由所有玩家购买彩票的金币累积而成，系统也会每轮自动注入基础奖金。
- **中奖分成**：中奖者将获得奖池中 90% 的金币（扣除 10% 税费）以及额外的 10 级经验奖励。

## ⌨️ 常用指令
| 指令 | 功能 |
| :--- | :--- |
| `/lottery buy <数量>` | 购买彩票（每轮限 100 张） |
| `/lottery status` | 查看奖池金额、剩余时间和参与人数 |
| `/lottery winners` | 查看最近的中奖名单 |

::: tip 小提示
每购买一次彩票有 60 秒的冷却时间，记得提前规划好你的购买量。
:::
