# 指令列表

以下是常用的基础指令。

## 位置与传送

### 1. 主城与出生点
- `/spawn`: 瞬间回到服务器主城。这里是贸易中心、任务发布点以及旅者的集散地。
- `/sethome [名称]`: 在当前位置设置一个家。您可以设置多个家（如：`/sethome farm`, `/sethome base`）。
- `/home [名称]`: 回到指定的家。

### 2. 玩家间传送 (TPA)
- `/tpa [玩家名]`: 请求传送到某位玩家身边。
- `/tpahere [玩家名]`: 请求某位玩家传送到您身边。
- `/tpaccept`: 接受他人的传送请求。
- `/tpdeny`: 拒绝他人的传送请求。

### 3. 随机传送
- `/rtp`: 将您随机传送到主世界的某个角落。适合寻找无人开垦的处女地。

## 基础交互与信息
- `/suicide`: 自杀（用于卡死或迷路时的极端自救）。
- `/msg [玩家名] [消息]`: 向某位玩家发送私聊。
- `/list`: 查看当前在线的旅者名单。
- `/ping`: 查看您与服务器之间的即时延迟（单位：毫秒）。

## 经济基础
- `/money` 或 `/bal`: 查看您的星币余额。
- `/pay [玩家名] [金额]`: 向他人转账。

**提示**: `[]` 内为变量，输入时不要带括号。大部分指令支持 Tab 键自动补全。
