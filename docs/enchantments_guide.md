# 附魔功能指南

欢迎使用 Enchantio 附魔系统！本指南将为您详细介绍服务器中新增的自定义附魔功能，包括效果说明、获取方式及兼容性。

## 附魔总览

本服务器共引入了 69 种全新附魔，分为 **通用附魔**、**战斗附魔**、**工具附魔** 和 **诅咒附魔** 四大类。

### 快速跳转列表
- [战斗类附魔](#战斗类附魔) - 提升伤害、防御或特殊战斗效果。
- [工具与实用类附魔](#工具与实用类附魔) - 自动化采集、经验加成等。
- [弓弩类附魔](#弓弩类附魔) - 增强远程武器性能。
- [防具类附魔](#防具类附魔) - 提升生存能力。
- [诅咒附魔](#诅咒附魔) - 带有负面效果的特殊附魔。

## 互斥规则说明

::: info 提示
某些附魔由于效果冲突，无法同时存在于同一件物品上。
:::

- **防御类互斥**：`安全气囊`、`绝境`、`壁垒` 与原版的 `保护`、`爆炸保护` 等互斥。
- **攻击类互斥**：`血蚀`、`影袭`、`驱散`、`断首` 与原版的 `锋利`、`亡灵杀手`、`节肢杀手` 互斥。
- **元素类互斥**：`焦灼` 与 `霜刃` 互斥，且都与 `火焰附加` 互斥。
- **远程类互斥**：`远袭`、`定准`、`静止` 与 `力量` 互斥；`回矢` 与 `无限` 互斥；`弹返` 与 `多重射击` 互斥。
- **采集类互斥**：`精研`、`洞察` 与 `精准采集` 互斥。
