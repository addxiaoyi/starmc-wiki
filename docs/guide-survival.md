# 生存指南

StarMC 以自由探索为主，以下建议供参考。

## 🪐 自由降临
StarMC 不设繁琐的强制任务。您可以选择在主城结识新朋友，也可以直接开启属于您的冒险。
- **随心传送**：使用 `/rtp` 指令，随机降落在万格之外的未知荒野。
- **寻找心仪之地**：高山、深海或是茂密丛林？一切由您决定。

## 🏡 建立您的专属领域
您可以独自隐居，也可以邀请志同道合的伙伴共同建设。
- **圈地保护**：攒够一定的星币后，使用 `/res create` 保护您的个人建筑和箱子，确保您的创作不受干扰。
- **个性化表达**：利用服务器丰富的表情系统和个性标识，展现独特的自我。

## 🌾 多样化的生活方式
在这里，生存质量取决于您的想象力，而非单纯的数值。
- **佛系务农**：建立自动化的农场，在清晨享受丰收的喜悦。
- **经商致富**：将您的产出通过自助贸易商店出售给其他玩家。
- **技能专精**：通过挖掘、狩猎或钓鱼提升个人属性，解锁更多趣味玩法。

## 🤝 融入温暖的社区
StarMC 最核心的价值在于人与人的链接。
- **加入公会**：寻找与您志趣相投的团体，共同参与社区建设。
- **互助分享**：在公屏打个招呼，您会发现这里的玩家非常热衷于分享经验与物资。
- **共同创造**：每个建筑都是社区历史的一部分。

::: tip 探索提示
`/sethome [名称]` 可以随时标记您的家。在 StarMC，迷路也是一种有趣的冒险，但记得给自己留个回家的标记。
:::
