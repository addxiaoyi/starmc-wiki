# 个性化标识系统

在 StarMC，你可以通过各种方式获得独特的头衔、勋章和称号，让你在聊天框和头顶显示上与众不同。

## 🏆 称号管理
我们提供了一个直观的界面来管理你的所有称号。

- **打开仓库**：输入 `/zprefix` 或 `/prefix` 即可打开称号界面。
- **佩戴与切换**：在界面里点击你已解锁的称号，聊天框和头顶显示会立即更新。
- **获取途径**：完成成就、参与社区活动、提升技能等级或者完成特定任务都能拿到新称号。
- **永久保留**：一旦解锁，称号将永久保存在你的账号中，重新登录也不会丢失。

## ✨ 进阶显示
- **动态前缀**：部分稀有称号带有华丽的动态效果或粒子特效。
- **实时预览**：在界面底部可以查看你当前的解锁进度和称号总数。

## ⌨️ 常用指令
| 指令 | 功能 |
| :--- | :--- |
| `/prefix` | 打开称号管理界面 |
| `/zprefix` | 打开详细的称号仓库 |
| `/np open` | 调整其他个性化显示设置 |

::: tip 小提示
有些称号是以“卷轴”形式发给你的，记得先在背包里右键激活，然后去 `/prefix` 菜单里佩戴。
:::
