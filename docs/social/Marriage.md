# 伴侣与婚姻系统

在 StarMC，可以与心仪的伙伴建立婚姻关系，享受专属的互动功能和共同成长的福利。

## 💍 建立羁绊
服务器的婚姻系统需要一位“牧师”作为见证人。

- **牧师主持**：结婚仪式必须由拥有 **牧师 (Priest)** 身份的玩家主持。
- **唯一性**：每位玩家只能拥有一位伴侣，珍惜眼前的伙伴。
- **全服公告**：新人的喜讯会向全服播报，分享你们的幸福时刻。

## ✨ 专属特权
建立关系后，你将解锁一系列便捷功能：
- **快速传送**：输入 `/marry tp` 即可立即到达伴侣身边。
- **私密聊天**：使用 `/marry chat` 开启两人的专属频道。
- **经验加成**：当你们在一定范围内一起行动时，会获得额外的经验奖励。
- **状态提醒**：伴侣上线或下线时，你都会收到温馨的提示。

## ⌨️ 常用指令
输入 `/marry` 即可打开婚姻主菜单，查看所有相关操作和信息。

::: tip 小提示
经营一段关系需要耐心，定期互动和赠礼可以增加你们的亲密度，解锁更多高级功能。
:::
