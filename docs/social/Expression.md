# 互动表情与图片

在 StarMC，使用丰富的图片表情、动态 GIF 甚至自定义展示框，让社交互动更加生动有趣。

## 😄 图片表情
- **快捷输入**：在聊天框输入关键词（如 `:smile:`）即可转换为对应表情。
- **智能补全**：输入关键词后按 **TAB** 键，系统会弹出匹配的表情建议。
- **全场景使用**：表情不仅能发在聊天框，还能用在铁砧重命名、木牌书写和私聊中。
- **资源包支持**：表情依赖服务器资源包显示，如果显示为乱码，请尝试输入 `/emojis update` 更新。

## 🖼️ 图片展示框
- **管理菜单**：输入 `/if menu` 管理你的所有图片展示框。
- **自定义展示**：可以将自己的图片或精美海报上传到游戏内的展示框中展示。

## ⌨️ 常用指令
| 指令 | 功能 |
| :--- | :--- |
| `/emojigui` | 打开表情库选择器 |
| `/if menu` | 管理你的图片展示框 |

::: tip 小提示
如果发现表情显示异常，请先检查你的材质包是否已正确应用（详见基础支撑 -> 自定义资源加载）。
:::
